Validation Workflow

Simple Testing and Verification

Based on your email request: run safely, verify correctly, document results.

1) Develop or collect scripts

Scripts 01 to 06 cover breach detection, Windows/Sysmon-style analysis, triage, phishing, networking, and SIEM concepts.

2) Test in safe virtual lab

Run only in authorized VM/lab. Never run on unauthorized or production systems.

3) Verify each script

Check command return code, output file creation, and result content (alerts/findings/incidents/events).

4) Prepare short report

For each script: what it does, how it works, success status, and errors/improvements needed.

Required Policy

All tests must be performed only in authorized virtual lab environments for learning and research.
